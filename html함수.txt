[<body> 태그 내부에 들어가는 대표적인 태그들]
<br> 줄바꿈 유사함수 <pre> 태그 (줄바꿈 및 띄어쓰기,탭 등 다 출력됨)
<p> 하나의 문단을 만들 때 쓰임
<b> 글자를 굵게 표시함 유사함수 <strong> 태그
<i> 글자를 기울여서 표시함
<h#> h1,h2,h3,h4,h5,h6 태그 글자의 크기 1이 제일큼
<a> 하이퍼링크 걸어주는 태그 href : 클릭시 이동 할 링크, target : 링크를 여는 방법
	ex ) <a href="http://google.co.kr" target="_blank">Go Google (new window)</a>
<img> 이미지를 삽입하는 태그 src 속성을 통해 이미지 경로를 지정 width 가로크기 height 세로크기
	ex ) <img src="my_profile.png" width="500" height="300">
<table> 태그는 HTML 문서에서 표를 만드는 태그 행과 열을 위해 <tr>(행),<td>(열) 등의 태그도 함께 사용
<div> 가상의 레이아웃을 설계하는데 쓰임 주로 css과 연동함 display 속성 block 줄바꿈 가능
<span> <div> 태그처럼 특별한 기능 x css와 함께 쓰임 display 속성이 inline 줄바꿈 x
<li> 목록을 만드는 태그 단독으로 쓰이지 않음 <ul> <ol> 내부에 들어감 // 메뉴를 만들때도 사용
이중 리스트를 만들기 위해서는 <ul> 이나 <ol> 하위에 다시 <ul> 이나 <ol> 태그를 사용
<ol> 번호를 매겨 순서가 있는 목록을 만듦
<ul> 순서없이 모양으로 목록을 만듦
